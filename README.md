# 地图可视化库inmap应用

使用vue+inmap做的页面

注意事项：

1.inmap要通过npm下载 然后在vue组件中import进来；如果直接写在index.html的头部 有些功能貌似不行

2.城市坐标获取的api中的ak是在这里申请的 https://lbsyun.baidu.com/index.php?title=%E9%A6%96%E9%A1%B5，失效了需重新申请。有些城市的坐标貌似获取不了。
